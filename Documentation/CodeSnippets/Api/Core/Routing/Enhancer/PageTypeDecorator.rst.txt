.. php:class:: PageTypeDecorator

   Resolves a static list (like page.typeNum) against a file pattern. Usually added on the very last part
   of the URL.
   
   It is important that the PageType Enhancer is executed at the very end in your configuration, as it modifies
   EXISTING route variants.
   
   routeEnhancers:
     PageTypeSuffix:
       type: PageType
       default: ''
       index: 'index'
       map:
         '.html': 1
         'menu.json': 13
   
   See source code on `GitHub <https://github.com/TYPO3/typo3/blob/main/typo3/sysext/core/Classes/Routing/Enhancer/PageTypeDecorator.php>`__.
