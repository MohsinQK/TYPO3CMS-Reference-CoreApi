   .. php:method:: convertFlexFormContentToArray(string $flexFormContent, string $languagePointer, string $valuePointer)
   
      Parses the flexForm content and converts it to an array
      The resulting array will be multi-dimensional, as a value "bla.blubb"
      results in two levels, and a value "bla.blubb.bla" results in three levels.
      
      Note: multi-language flexForms are not supported yet
      
      See source code on `GitHub <https://github.com/TYPO3/typo3/blob/main/typo3/sysext/core/Classes/Service/FlexFormService.php#L37>`__.
      
      :param string $flexFormContent: flexForm xml string
      :param string $languagePointer: language pointer used in the flexForm
      :param string $valuePointer: value pointer used in the flexForm
