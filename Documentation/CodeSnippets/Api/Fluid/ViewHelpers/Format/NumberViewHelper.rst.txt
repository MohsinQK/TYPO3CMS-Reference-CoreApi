.. php:class:: NumberViewHelper

   Formats a number with custom precision, decimal point and grouped thousands.
   
   See https://www.php.net/manual/function.number-format.php.
   
   Examples
   ========
   
   Defaults
   --------
   
   ::
   
      <f:format.number>423423.234</f:format.number>
   
   ``423,423.20``
   
   With all parameters
   -------------------
   
   ::
   
      <f:format.number decimals="1" decimalSeparator="," thousandsSeparator=".">
          423423.234
      </f:format.number>
   
   ``423.423,2``
   
   See source code on `GitHub <https://github.com/TYPO3/typo3/blob/main/typo3/sysext/fluid/Classes/ViewHelpers/Format/NumberViewHelper.php>`__.
