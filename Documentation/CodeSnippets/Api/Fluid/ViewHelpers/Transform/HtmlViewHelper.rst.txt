.. php:namespace::  TYPO3\CMS\Fluid\ViewHelpers\Transform

.. php:class:: HtmlViewHelper

   Transforms HTML and substitutes internal link scheme aspects.
   
   Examples
   ========
   
   Default parameters
   ------------------
   
   ::
   
      <f:transform.html selector="a.href" onFailure="removeEnclosure">
        <a href="t3://page?uid=1" class="home">Home</a>
      </f:transform.html>
   
   Output::
   
       <a href="https://example.com/home" class="home">Home</a>
   
   Inline notation
   ---------------
   
   ::
   
      {content -> f:transform.html(selector:'a.href', onFailure:'removeEnclosure')}
   
   See source code on `GitHub <https://github.com/TYPO3/typo3/blob/main/typo3/sysext/fluid/Classes/ViewHelpers/Transform/HtmlViewHelper.php>`__.
